## Description

This API allows you to retrieve a logo associated with a specific wallet. It fetches the logo data from a cache or, if not available, retrieves it from a media repository. The logo is then sent to the client.

### Endpoint
`POST /api/qr-logo`

This endpoint will fetch the logo of the wallet linked to the current user's `WalletId` or `ApiKeySecretId`. If either of these values is missing, an error will be returned.

### Error Handling

If the `ApiKeySecretId` or `WalletId` is not available for the current user, the API will return a `UserFriendlyException` with the message `"Kullanıcı Bulunamadı"`. If the wallet is not found, the message `"Girdiğiniz bilgiye ait cüzdan bulunamadı."` will be returned.
